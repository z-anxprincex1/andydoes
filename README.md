# andydoes

Personal portfolio site for Anand Prince Purty.

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Local development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Preview production build

```sh
npm run preview
```

## Notes

- The site includes portfolio content, experience, projects, and achievements.
- Environment-specific values can be configured through local `.env` files when needed.
